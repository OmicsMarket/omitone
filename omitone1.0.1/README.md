# 欧米通 — 学习通自动完成任务点

## 核心思想

学习通课程任务点分三类，欧米通逐一自动处理：

| 类型 | 内容 | 处理方式 |
|------|------|----------|
| **A 视频** | 有任务点的视频 | 先尝试拖到底秒过 → 拖不动则正常倍速播放 → 播完自动切下一个 |
| **B 题目** | 单选题/多选题/判断题/填空题/简答题 | 配置 API Key 后 AI 自动答题 → 填入答案 |
| **C 其他** | 纯文本/讨论/无任务点视频 | 2 秒内直接跳过 |

## 安装

1. 打开 Chrome/Edge → 地址栏输入 `chrome://extensions`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择 `xuexitong-extension/` 文件夹
5. 点击扩展图标 → 配置 API Key → 保存

## 使用

1. 打开学习通课程页面 → 自动弹出控制面板
2. 点击 **开始** → 全自动运行
3. 开关可随时切换：自动连播 / AI 答题

## 架构

```
popup.html      设置面板（API Key、倍速、开关）
background.js   Service Worker（中转 MiniMax/Claude API 调用）
content.js      ISOLATED world（注入桥接 + SPA 导航检测）
page.js         MAIN world（主逻辑，2 秒 tick loop）
```

### 2 秒 Tick Loop（`page.js`）

```
每 2 秒执行一次 _tick()：

  1. 检测页面切换 → URL 变化则重置状态
  2. 正在播放中 → 跳过（视频监控保活）
  3. 找视频 → 有 → 检查任务点 → 拖到底 or 正常播放
  4. 找测验 → 有 → AI 答题 or 跳过
  5. 尝试切换"视频"页签
  6. 已完成 → 跳过
  7. 纯文本 → 跳过下一节
```

## 机制细节

### 视频秒过（拖动进度条）

```
等元数据加载
  → seek 到 dur - 0.5s，反复 5 次突破防护
  → 等 3 秒确认进度条没弹回
  → 停住了 → 秒过 ✅
  → 弹回了 → 正常 1x 播放 ✅
```

### 视频保活

每秒检查：暂停了 → 恢复播放 / 进度卡住 7s → 恢复 / 恢复过频 → 停止干预

### 多视频串联

同一节多个视频：播放完自动切下一个 `[1/3] → [2/3] → [3/3]`，全部播完切下一节。

### 连续跳过

纯文本或已完成章节自动点"下一节"，跳过上限 500 节防死循环。

### 弹窗处理

自动关闭"任务点未完成，是否去完成？"等拦截弹窗。

### SPA 导航

Hook `history.pushState/replaceState` + `popstate`，页面切换时自动重新注入脚本。旧实例通过 `_XXT_UID` 自动停摆。
